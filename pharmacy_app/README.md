# Pharmacy Mobile App

Flutter mobile application for pharmacies to receive prescription requests and create quotes.

## Features

- Pharmacy authentication
- Receive prescription requests from nearby patients
- View prescription images
- Create detailed price quotes with medicine breakdown
- Manage order preparation
- Track earnings and statistics
- Push notifications for new requests
- Order history

## Screens

1. **Splash Screen**
2. **Login**
3. **Dashboard** - Overview of requests and orders
4. **Prescription Requests** - List of nearby requests
5. **Prescription Viewer** - View prescription image
6. **Create Quote** - Add medicines and prices
7. **Order Preparation** - Manage active orders
8. **Order History**
9. **Earnings Dashboard**
10. **Profile & Settings**

## Setup

```bash
flutter pub get
flutter run
```

## Configuration

Update API URL in `lib/services/api_service.dart`
Add Firebase and Google Maps configuration files
