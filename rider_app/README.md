# Rider Mobile App

Flutter mobile application for delivery riders to accept and complete deliveries.

## Features

- Rider authentication
- View nearby delivery requests
- Accept delivery assignments
- Navigate to pharmacy for pickup
- Navigate to patient for delivery
- Update delivery status
- Track earnings
- Delivery history
- Real-time location updates

## Screens

1. **Splash Screen**
2. **Login**
3. **Dashboard** - Available deliveries
4. **Delivery Requests** - List of nearby orders
5. **Delivery Details** - Order information
6. **Navigation** - Google Maps integration
7. **Pickup Confirmation**
8. **Delivery Confirmation**
9. **Earnings Dashboard**
10. **Delivery History**
11. **Profile**

## Setup

```bash
flutter pub get
flutter run
```

## Configuration

Update API URL in `lib/services/api_service.dart`
Add Firebase and Google Maps configuration files
